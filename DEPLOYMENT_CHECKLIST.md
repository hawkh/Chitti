## 🚀 **Deployment Commands**

### Quick Start (Docker)
```bash
# Clone repository
git clone <repository-url>
cd chitti-ai-ndt

# Start services
npm run docker:up

# Initialize database
npm run setup

# Access application
http://localhost:3000
```

### Manual Setup
```bash
# Install dependencies
npm install

# Setup database
npm run db:migrate
npm run db:generate

# Initialize system
npm run setup

# Start development
npm run dev
```
